# SmartCampusAPI

A RESTful API built with JAX-RS (Jersey) and Grizzly HTTP server for managing rooms and sensors across a smart university campus.

---

## API Overview

This API allows campus facilities managers to manage Rooms and Sensors deployed across the campus. It supports full CRUD operations for rooms, sensor registration and filtering, historical sensor readings, and robust error handling.

Base URL: `http://localhost:8080/api/v1`
